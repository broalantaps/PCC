# Here is the repo of *Pre-training Context Compressor for Large Language Models*.

## Preparation
- Export HuggingFace Token if you want to upload model to 🤗HuggingFace:
```bash
export HF_TOKEN=your_token_here
```
- Download Dataset and Pre-training checkpoint: **Pre-trained checkpoints will be released upon acceptance of this paper to benefit research in this direction.**

## Running the Project
```bash
bash main.sh
```
